 						KALKULATOR MACIERZOWY 
Dzieki niemu mozesz dodac, odjac a nawet pomnozyc macierze przez siebie.

Aby uzyskac wynik:
->Wypelnii pliki:
*M1.txt - pierwsza macierz(wpisywana w jedna linie);
*M2.txt - druga macierz(wpisywana w jedna linie);
*S1.txt - rozmiar pierwszej macierzy(wpisywana ilosc wierszy po spacji iloszc kolumn (np.3 3 to jest macierz rozmiaru 3x3));
*S2.txt -rozmiar drugiej macierzy.
-> Uruchom program.
-> Wpisz rodzaj operacji jaka chcesz uzyskac(dodac, odjac lub pomnozyc).
-> Macierze automatycnie pobierane z plikow.
-> Dostajesz wynik !:).
